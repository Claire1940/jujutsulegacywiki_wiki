<Section icon={Swords} title="Analisis Moveset dan Strategi Pertarungan">

Loop pertarungan untuk **jujutsu legacy mechamaru v2** berkisar pada menjaga jarak sambil melakukan siklus proyektil berdampak tinggi. ### Skill Utama dan Scaling Damage

Tabel berikut membedah kemampuan inti yang tersedia dalam kit V2. Angka damage didasarkan pada karakter dengan 200 poin dalam statistik Cursed Energy.

| Nama Skill | Deskripsi | Cooldown | Tipe Damage |
| :--- | :--- | :--- | :--- |
| **Miracle Cannon** | Menembakkan sinar energi terkonsentrasi. | 6 detik | Burst / Piercing |
| **Pigeon Viola** | Meluncurkan beberapa rudal pelacak. | 10 detik | Pelacakan / Multi-hit |
| **Albatross** | Ledakan overhead masif dengan AoE luas. | 15 detik | Berat / Knockback |
| **Shield Deploy** | Membuat penghalang sementara. | 20 detik | Defensif / Buffer |
| **Absolute Mode** | Buff sementara untuk semua output damage. | 60 detik | Transformasi |

### Rantai Kombo yang Dioptimalkan

Untuk memaksimalkan DPS Anda, gunakan urutan berikut:
- **Membersihkan Mob**: Pigeon Viola -> Albatross -> Miracle Cannon.
- **Burst Bos**: Absolute Mode -> Miracle Cannon -> Pigeon Viola (Ulangi).
- **Pertahanan PvP**: Shield Deploy -> Dash Mundur -> Pigeon Viola.

<Callout type="success" title="Kesuksesan Strategi">
