<Section icon={Target} title="Como Obter o Mechamaru V2">

Desbloquear o **jujutsu legacy mechamaru v2** é um processo de várias etapas que exige um investimento significativo de tempo e sorte com drops de chefes. Você não pode simplesmente comprar esta evolução; ela deve ser conquistada através de marcos específicos de progressão. A maioria dos jogadores gasta aproximadamente 10 a 15 horas de grinding focado para reunir os materiais necessários após atingir o limite de nível.

<Steps>
  <Step num="1" title="Alcançar Requisito de Nível">
    Você deve atingir o Nível 150. Este é o limite obrigatório para iniciar a linha de missões de evolução. Recomendamos farmar na zona "Cursed Forest" para os ganhos de XP mais rápidos em 2026.
  </Step>
  <Step num="2" title="Adquirir o Núcleo Mecânico">
    Derrote o **Chefe Marionete de Kyoto** localizado no workshop secreto. A taxa de drop para o "Núcleo Mecânico" (Mechanical Core) é de aproximadamente 5%. Este item é o componente base para o chassi V2.
  </Step>
  <Step num="3" title="Coletar Sucata Amaldiçoada">
    Reúna 50 unidades de Sucata Amaldiçoada (Cursed Scrap) desmontando ferramentas amaldiçoadas de grau Raro ou derrotando inimigos do tipo mecânico no distrito urbano.
  </Step>
  <Step num="4" title="O Ritual de Evolução">
    Leve todos os materiais ao NPC "Kokichi's Shadow" no mapa da Escola de Kyoto. Pague 500.000 Yen para finalizar a transformação no Mechamaru V2.
  </Step>
</Steps>

| Material | Quantidade | Fonte | Taxa de Drop |
| :--- | :--- | :--- | :--- |
| **Núcleo Mecânico** | 1 | Chefe Marionete de Kyoto | 5% |
| **Sucata Amaldiçoada** | 50 | Inimigos Mecânicos | 25% |
| **Yen** | 500.000 | Missões/Tarefas Diárias | 100% |
| **Orbe de Energia Amaldiçoada** | 10 | Espíritos de Alto Grau | 15% |

<Callout type="warning" title="Aviso">
O processo de evolução é permanente. Uma vez que você atualiza para o V2, não pode reverter para o V1 sem usar uma Pedra de Reset de Técnica, que custa Robux ou é um drop raro de chefes mundiais.
</Callout>

<Section icon={Swords} title="Análise de Moveset e Estratégia de Combate">

O ciclo de combate do **jujutsu legacy mechamaru v2** gira em torno de manter a distância enquanto alterna entre projéteis de alto impacto. Seu objetivo é manter os inimigos atordoados ou empurrados para trás para que nunca consigam fechar a distância. No PvP, isso requer precisão no tempo; no PvE, exige entender os padrões dos chefes para desferir seu "Canhão Supremo" durante as janelas de vulnerabilidade.

### Habilidades Principais e Escalonamento de Dano

A tabela a seguir detalha as habilidades principais disponíveis no kit V2. Os números de dano são baseados em um personagem com 200 pontos no atributo de Energia Amaldiçoada.

| Nome da Habilidade | Descrição | Recarga | Tipo de Dano |
| :--- | :--- | :--- | :--- |
| **Miracle Cannon** | Dispara um feixe concentrado de energia. | 6s | Explosão / Perfurante |
| **Pigeon Viola** | Lança vários mísseis teleguiados. | 10s | Rastreamento / Multi-hit |
| **Albatross** | Uma explosão aérea massiva com enorme AoE. | 15s | Pesado / Repulsão |
| **Shield Deploy** | Cria uma barreira temporária. | 20s | Defensivo / Buffer |
| **Absolute Mode** | Bônus temporário para todo o dano causado. | 60s | Transformação |

### Cadeias de Combo Otimizadas

To maximize your DPS, use the following sequences:
- **Limpeza de Hordas**: Pigeon Viola -> Albatross -> Miracle Cannon.
- **Explosão em Chefes**: Absolute Mode -> Miracle Cannon -> Pigeon Viola (Repetir).
- **Defesa PvP**: Shield Deploy -> Dash para trás -> Pigeon Viola.

<Callout type="success" title="Sucesso na Estratégia">
Usar "Pigeon Viola" imediatamente após o "Absolute Mode" resulta em um aumento de 25% no dano para cada míssil, tornando-se a maneira mais eficiente de drenar a barra de vida de um chefe.
</Callout>
